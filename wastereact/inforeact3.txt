1ï¸âƒ£ What is useState?

useState is a React hook that lets you add state (data that changes) to a functional component.

Without state, a React function component is static.
With useState, you can store values and update them when something happens (like a button click).

2ï¸âƒ£ How to use it

  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

3ï¸âƒ£ What useState returns

useState(initialValue) returns an array with 2 items:

value â†’ the current state

setValue â†’ a function to update the state

Example:

count is 0 initially.

When you call setCount(5), React updates count to 5 and re-renders the component.

4ï¸âƒ£ How state updates

You donâ€™t change count directly (like count = 5), because React wonâ€™t know it changed.

You always use the setter function (setCount) to tell React: "Hey, update this value and refresh the UI."

import React, { useState } from "react";

function UserForm({ refresh }) {
  const [username, setUsername] = useState("");
  const [password, setPassword] = useState("");

  async function handleAddUser() {
    await fetch("/users", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ username, password }),
    });
    setUsername("");
    setPassword("");
    refresh(); // reload users in App
  }

  return (
    <div>
      <h2>Add User</h2>
      <input
        type="text"
        placeholder="username"
        value={username}
        onChange={(e) => setUsername(e.target.value)}
      />
      <input
        type="password"
        placeholder="password"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
      />
      <button onClick={handleAddUser}>Add User</button>
    </div>
  );
}
